1️⃣ Difference between var, let, and const?

> > Var is old version, Not used anymore.
> > Let is changable, Used in Modern JS.
> > Const is unchangeable, Used in modern JS.

2️⃣ What is the spread operator (...)?

> > The spread operator expands elements from an array or properties from an object into another array or object.

3️⃣ What is the difference between map(), filter(), and forEach()?

> > map() Creates a new array.
> > filter() Creates a new array with elements that pass a condition.
> > forEach() Executes a function for each element.

4️⃣ What is an arrow function?

> > An arrow function is the shorter version of functions in JavaScript.

5️⃣ What are template literals?

> > Template literals allow embedding variables inside strings using backticks ( ) and ${}.
